# Free Quality Upgrade Plan

## The honest answer

All free + fully automatic + long videos = yes.
All free + fully automatic + premium 3D animation = no.

Your free quality ceiling improves by using reusable assets, not by regenerating everything.

## Best free upgrades

### 1. Reusable character PNGs
Create fixed transparent PNGs for:
- Benny Fox
- Luna Bunny
- Milo Bear
- Coco Cat
- Olive Owl

Put them in:
`assets/characters`

### 2. Reusable backgrounds
Create or download free backgrounds:
- forest night
- sunny meadow
- rainbow classroom
- sleepy bedroom
- river
- star sky

Put them in:
`assets/backgrounds`

### 3. Free overlays
Download free loop videos from Pixabay/Pexels:
- rain
- snow
- stars
- fog
- clouds

Then later add them as overlays.

### 4. Voice quality
Edge TTS is already used.
Best free voices:
- en-US-AriaNeural
- en-US-AnaNeural
- en-US-ChristopherNeural
- en-US-JennyNeural

### 5. Keep videos private first
Do not publish automatically as public until the system is stable.
