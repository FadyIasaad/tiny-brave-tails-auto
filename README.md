# Tiny Brave Tails Auto Studio V2

A free, one-click GitHub Actions video automation system for kids content.

## What this repo does

Each button in GitHub Actions can:

1. Generate a video idea
2. Generate an English script
3. Generate free Edge TTS voiceover
4. Create simple animated visuals with zoom/pan/particles
5. Render a video
6. Upload to YouTube
7. Add the video to the correct playlist
8. Save metadata locally

## Free stack

- GitHub Actions
- Python
- Edge TTS
- MoviePy
- Pillow
- FFmpeg
- YouTube Data API

## Important truth

This is a free automated production engine.
It is NOT Pixar, Doggyland, Runway, Kling, or premium AI animation.

Quality level:
- simple
- clean
- consistent
- scalable
- suitable for testing and uploading

The biggest upgrade path is adding your own reusable character PNGs and backgrounds inside `assets/`.
